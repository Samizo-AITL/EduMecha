# 🔩 03 - Shaft Mount Intro

この演習では、**軸受けと取り付け面構造の基本モデリング**を通じて、対称構造・複数フィーチャの組み合わせ・拘束の概念を学習します。

## 🧭 学習目標

- 軸部と基板部の合成形状モデリング
- 円形拘束と基準面の活用
- 対称性を保ったスケッチと拘束設計
- 設計意図に基づく形状の展開
