# 🔧 01 - Block with Holes

この演習では、**直方体に対する基本的な穴加工**を通じて、パラメトリック3Dモデリングの初歩を学びます。

## 🧭 学習目標

- スケッチと押し出し機能の使い方
- 穴の作成（カット or 穿孔）
- 寸法拘束（サイズ・位置）の適用
- デザイン意図に沿ったモデル変更
