# 🧱 04 - Plate with Pocket

この演習では、**プレート形状へのポケット加工（くぼみ・段差）**を通じて、スケッチ拘束・押し出しカット・パターン配置の基本を学びます。

## 🧭 学習目標

- 押し出しカットと押し出しボスの使い分け
- 内部ポケット構造の作成と拘束配置
- 複数スケッチの整合と整列
- パターン・ミラー機能の基本活用
