# 📐 02 - L-Bracket Design

この演習では、**L字型ブラケットのパラメトリック設計**を通じて、複数押し出し・拘束条件の活用方法を学習します。

## 🧭 学習目標

- スケッチ面の選択と基準の考え方
- 複数方向の押し出しと合成形状
- 寸法拘束と関係拘束の適用
- 対称性や変更容易性を意識した設計
