# 03. Gimbal Frame

この演習では、2軸ジンバル機構の外枠（フレーム）設計を通じて、動的機構部品の構造設計を学びます。センサ（IMU等）やカメラの姿勢制御に用いられる基本構造を対象とします。

## 🧰 学習ポイント

- 自由度を確保する軸構成の理解（Pitch / Yaw）
- 回転中心・荷重中心を意識したフレーム設計
- サーボ取付とケーブルマネジメントの設計配慮
- 複数部品間の組立関係（同軸拘束・回転可動）
